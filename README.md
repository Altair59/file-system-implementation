# file-system-implementation
A file system implementation using ext2. 
Supported operations: 
* mount and unmount the file system in the designated location
* ls, mkdir, rm
* touch to make a new file
* cat to see the content of a file
* stat to show the status of a file / directoty
* mv to change name and location of a file / directory
* copy and write data
* truncate to increase or decrease file size
